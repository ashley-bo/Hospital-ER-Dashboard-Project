# Hospital-ER-Dashboard-Project
This Power BI report provides an interactive overview of Emergency Room performance metrics across three pages:

Monthly View — Tracks daily trends for patient volume, average wait time, satisfaction scores, and referrals for a selected month, with breakdowns by admission status, gender, and patient arrival by day/hour.
Consolidated View — Presents the same KPIs aggregated at a monthly level across a configurable date range, enabling trend comparison over time.
Patient Details — A filterable table showing individual patient records including name, age, gender, race, wait time, department referral, and admission status.

Key metrics tracked: number of patients, average wait time, patient satisfaction score, number of referrals, and the percentage of patients seen within 30 minutes.
